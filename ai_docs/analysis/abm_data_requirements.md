# ABM 精度向上に必要なデータ要件

## 背景

現状の社会シミュレーション（ABM）は PoC レベルであり、パラメータが実測値に基づいていない。
インテージの保有データを活用することで、以下の3点で精度向上が見込める。

---

## 1. SCI（全国消費者パネル）— キャリブレーション用

### 必要なデータ
- 睡眠改善カテゴリ商品（ヤクルト1000、キリン免疫ケア等）の**月次購入率推移**
- 個人別のブランドスイッチ履歴（新商品受容性・採用閾値の算出用）

### 活用方法
- `calibration.py` の `calibrate(real_adoption_curve)` に月次採用率カーブを投入
- Nelder-Mead でKOL影響力・ピア影響力パラメータを最適化
- `AgentProfile` の `receptivity` / `adoption_threshold` を実データの連続値分布から算出

### 期待効果
- シミュレーション結果が実際の市場採用カーブと整合するようになる
- 現状の3段階(early/moderate/late)マッピングが連続値に精緻化される

---

## 2. i-SSP（シングルソースパネル）— 影響力パラメータの根拠

### 必要なデータ
- 睡眠改善商品に関する**広告接触（TV CM / Web / SNS）→ 認知 → 購入**の導線データ
- 「口コミ・知人の推薦で知った」vs「広告で知った」の比率

### 活用方法
- KOL影響力（`kol_influence`）とピア影響力（`peer_influence`）の比率を実測値で設定
- `EventScheduler` に投入する CM キャンペーンの `impact_magnitude` を定量化

### 期待効果
- 影響力パラメータがデフォルト値（kol=0.15, peer=0.05）ではなく実測に基づく
- マス広告と口コミの影響バランスが現実を反映する

---

## 優先度

| 順位 | データ | 効果 |
|---|---|---|
| 1 | SCI 月次購入率推移 | キャリブレーション（最大効果） |
| 2 | SCI 個人別ブランドスイッチ | プロファイル精緻化 |
| 3 | i-SSP 接触→購買経路 | 影響力パラメータの根拠 |
