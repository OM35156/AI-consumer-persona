# 用語集

## ビジネス用語

| 用語 | 説明 |
|---|---|
| IHC | インテージヘルスケア |
| Digital Twin | 生活者の AI ペルソナを構築し、購買行動をシミュレーションする仕組み |
| ペルソナ | AI が再現する生活者の属性・購買行動・情報接触傾向のプロファイル |
| ライフステージ（LifeStage） | 学生 / 独身勤労 / 既婚子なし / 既婚子育て中 / 子独立後 / 退職後の区分 |
| カテゴリプロファイル（CategoryProfile） | 商品カテゴリごとの利用ブランド・購買哲学・価格感度・新商品受容度 |
| 購買意向（PurchaseIntent） | 必ず買いたい / たぶん買う / 検討する / たぶん買わない / 買わない の5段階 |
| ブランド認知（BrandAwareness） | ACTIVE_USER / PAST_USER / AWARE_NOT_USED / UNAWARE の4段階 |
| 情報チャネル（InformationChannel） | TV CM / Web広告 / SNS / 口コミ / 店頭 / 雑誌 / インフルエンサー / 検索 / EC |
| プロモーション反応 | 広告・キャンペーン等の情報提供に対する購買意向の変化 |
| ブランド接触（BrandExposure） | 特定チャネル経由でブランドに触れた記録（日付・内容・接触後意向） |
| 回答スタイル（ScaleUsagePattern） | 尺度設問の回答傾向。EXTREME / MODERATE / ACQUIESCENT / BALANCED |
| 採用状態（AdoptionState） | ABM におけるエージェントの状態遷移。NOT_ADOPTED → CONSIDERING → ADOPTED |
| インフルエンサー | 周囲への影響力が高い生活者。ABM では高 influencer_score のエージェント |
| PoC | Proof of Concept（概念実証） |

## 技術用語

| 用語 | 説明 |
|---|---|
| LLM | Large Language Model（大規模言語モデル） |
| Embedding | テキストをベクトル表現に変換すること |
| RAG | Retrieval-Augmented Generation。検索結果をプロンプトに注入する方式 |
| 2階層 RAG | 層①セグメントプロファイル + 層②ベクトルDB検索で文脈を構築する本システムの方式 |
| k-anonymity | 匿名化手法。同一属性の個人が k 人以上存在することを保証 |
| JS divergence | Jensen-Shannon ダイバージェンス。分布間の類似度を測定 |
| Replication | 同一条件でのシミュレーション繰り返し回数 |
| ABM | Agent-Based Model。個々の生活者エージェントの相互作用を通じて集団現象を再現 |
| Factoid | データ根拠のある行動事実。ペルソナの一人称発言の裏付けに用いる |
