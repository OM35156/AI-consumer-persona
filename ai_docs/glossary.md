# 用語集

## ビジネス用語

| 用語 | 説明 |
|---|---|
| IHC | インテージヘルスケア |
| Digital Twin | 医師の AI ペルソナを構築し、行動をシミュレーションする仕組み |
| ペルソナ | AI が再現する医師の属性・診療行動・処方傾向のプロファイル |
| Impact Track | 医師パネルデータ（プロモーション反応・処方データ） |
| プロモーション反応 | MR 訪問・Web セミナー等の情報提供に対する医師の反応 |
| 処方意向 | 特定の薬剤を処方する意思・傾向 |
| PoC | Proof of Concept（概念実証） |

## 技術用語

| 用語 | 説明 |
|---|---|
| LLM | Large Language Model（大規模言語モデル） |
| Embedding | テキストをベクトル表現に変換すること |
| k-anonymity | 匿名化手法。同一属性の個人が k 人以上存在することを保証 |
| JS divergence | Jensen-Shannon ダイバージェンス。分布間の類似度を測定 |
| Replication | 同一条件でのシミュレーション繰り返し回数 |
