# AI-Consumer-Persona

LLM を活用した**生活者 AI ペルソナ**構築・シミュレーションシステム。

調査データから生活者のプロファイルを構築し、ブランド反応や購買意向をシミュレーションする。

## 概要

- **ペルソナ構築**: 調査データ（属性・購買行動・情報接触）から生活者ペルソナを自動生成
- **RAG 基盤**: セグメントプロファイル + データカードによるデータグラウンディング
- **シミュレーション**: プロモーション反応・サーベイ回答・自由対話を LLM で再現
- **ABM**: エージェントベースモデルによるブランド採用の波及効果予測
- **プレテスト**: 施策シナリオの定量評価（ポテンシャルモデル）

## 技術スタック

| カテゴリ | 技術 |
|---|---|
| LLM | Anthropic Claude API |
| データモデル | Pydantic v2 |
| ベクトルDB | Qdrant |
| ABM | Mesa 3.x |
| UI | Streamlit |
| API | FastAPI |
